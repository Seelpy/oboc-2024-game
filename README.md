# Docker File
Запустить игру с помощью docker
```bash
./bin/run-image.sh
```

Прекратить работу игры
```bash
./bin/stop-image.sh
```

# Docker Compose
Запустить игру с помощью docker-compose
```bash
./bin/run-compose.sh
```

Прекратить работу игры
```bash
./bin/stop-compose.sh
```
